# No-Code Platform – Production Deployment

This repository lets you deploy the latest **No-Code App** directly from Docker Hub.
It includes automatic image updates — no rebuilds or manual pulls required.

---

## Deployment system

You can deploy the platform on a local/personal computer or a server:

- Local compute: Running the app on your local or personal computer, go to [Docker Compose](./compose)
- Server: Hosting on a server, go to [Docker swarm](./swarm)
